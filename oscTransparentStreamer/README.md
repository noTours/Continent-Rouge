### How to

	1. Create an openframeworks project adding the following addons ofxOsc + ofxXmlSettings + ofxHttpUtils + ofxJSON.
	2. Copy and remplace main.cpp, ofApp.cpp and ofApp.h from the repository to the src folder of your project.
	3. Add the remaining files (noToursWalker.h, noToursWalker.cpp, VHPhttpGetJson.h, VHPhttpGetJson.cpp) to your project.
	4. Copy the data folder from the repository to the bin folder of your project.
	
	
### Requirements

openFrameworks 8.3
