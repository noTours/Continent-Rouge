### How to

	1. Create an openframeworks project adding the following addons ofxOsc + ofxXmlSettings + ofxSvg + ofxHttpUtils + ofxJSON + ofxModestMaps.
	2. Copy and remplace main.cpp, ofApp.cpp and ofApp.h from the repository to the src folder of your project.
	3. Add the remaining 30 files to your project.
			noToursLocation.cpp
			noToursLocation.h
			noToursProjects.cpp
			noToursProjects.h
			noToursSoundPoint.cpp
			noToursSoundPoint.h
			noToursWalker.cpp
			noToursWalker.h
			ofxTextBlock.cpp
			ofxTextBlock.h
			ofxTextSuite.h
			ofxXmlSettings.cpp
			VHPbutton.cpp
			VHPbutton.h
			VHPgetJson.cpp
			VHPgetJson.h
			VHPpoint.cpp
			VHPpoint.h
			VHPpolyline.cpp
			VHPpolyline.h
			VHProundedButton.cpp
			VHProundedButton.h
			VHProundedRectangle.cpp
			VHProundedRectangle.h
			VHPsendOSC.cpp
			VHPsendOSC.h
			VHPshape.cpp
			VHPshape.h
			VHPtrueTypeFont.cpp
			VHPtrueTypeFont.h
	4. Copy the data folder from the repository to the bin folder of your project.
	5. Remplace ofxXmlSettings.cpp file in your ofxXmlSettings/src/ projects folder
	
### Requirements

openFrameworks 8.3
